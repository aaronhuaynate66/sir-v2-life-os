# SIR V2 — Life Operating System

> **Objetivo raíz: Conseguir Paz.**

Sistema operativo cognitivo-relacional privado. Segundo cerebro. Mission Control de vida.

## Stack

Next.js 15 + React 18 + TypeScript + Tailwind + Framer Motion + Zustand

## Correr

npm install && npm run dev

Abrir: http://localhost:3000/dashboard

## Estructura

docs/ - 10 documentos del sistema
src/engines/ - 12 engines de lógica (self, relationship, goal, signal, memory, context, timing, peace, financial, biological, ai-brain, recommendation)
src/data/fixtures/ - Datos de ejemplo
src/types/ - TypeScript types
src/app/dashboard/ - Mission Control UI

## Filosofia

datos to señales to contexto to memoria to timing to recomendacion to accion
